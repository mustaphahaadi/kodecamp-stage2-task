
# KODECAMP Stage 2 Task - Bash Scripting

## Simple Bash To-Do List Manager

A lightweight command-line script to manage your daily tasks directly from the terminal.

## Features
* **View Tasks:** Displays all current tasks with line numbers.
* **Add Tasks:** Appends new tasks to a storage file.
* **Delete Tasks:** Removes specific tasks by their number.
* **Persistent Storage:** Saves everything to `~/todo.txt`.

## How to Run

1. Open your terminal and navigate to the script folder.
2. Make the script executable:
   ```bash
   chmod +x todo.sh
   ```
3. Run the script:
   ```bash
   ./todo.sh
   ```
